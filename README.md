Análisis de Datos: Programa de Lealtad de Aerolínea

Este proyecto consiste en un análisis integral de un conjunto de datos de una aerolínea, abarcando desde la limpieza inicial hasta la extracción de conclusiones mediante análisis estadístico y visualización.

Estructura del Proyecto
El trabajo se divide en tres fases principales:

Fase 1: Exploración y Limpieza: Identificación de valores nulos, duplicados y preparación del dataset.

Fase 2: Análisis Estadístico: Exploración descriptiva de los datos para entender el perfil del cliente.

Fase 3: Visualización: Creación de gráficos para responder preguntas de negocio y detectar patrones.

Tecnologías Utilizadas
Python: Lenguaje principal.

Pandas & Numpy: Manipulación de datos.

Seaborn & Matplotlib: Visualización de datos.

Jupyter Notebook: Entorno de desarrollo.

Fuentes de consulta:
- Apuntes de lecciones
- Consulta con profesor y compañeras
- Gemini y Notebook LM

Puntos Clave del Análisis
1. Limpieza de Datos
Se realizó una integración de archivos (merge) y se gestionaron los valores nulos. Por ejemplo, en la columna salary, se optó por la imputación mediante la mediana para evitar sesgos por valores extremos.

2. Perfil del Cliente
A través del análisis estadístico, se determinó que la base de datos se caracteriza por un cliente de perfil profesional, con estabilidad familiar y una participación equilibrada en el programa de lealtad.

Promedio de vuelos reservados: ~4.13 vuelos.

Salario promedio: ~77,879 unidades.

3. Visualización de Resultados
Se generaron visualizaciones para entender el comportamiento temporal y demográfico.

Ejemplo: Se analizó la distribución de vuelos reservados por mes para identificar picos de demanda durante el año.

Conclusiones:
El conjunto de datos resultante está optimizado y permite crear estrategias de retención dirigidas, gracias a la homogeneidad en sectores como el estado civil y la educación de los clientes.
